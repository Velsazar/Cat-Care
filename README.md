## Descripción

GatoCare es una página web interactiva y moderna dedicada al cuidado de los gatos. Desarrollada por **D&S Inc**, esta página ofrece información completa sobre alimentación, salud, comportamiento y mucho más, con herramientas interactivas que hacen el aprendizaje divertido y fácil.

---

## Características Principales

### Secciones de Contenido
- **Inicio** - Hero section impactante con animaciones
- **About** - Información sobre D&S Inc y misión
- **Calculadora de Edad** - Convierte la edad de tu gato a años humanos
- **Test de Conocimientos** - Quiz interactivo de 6 preguntas
- **Mitos vs Realidades** - Desmiente creencias falsas sobre gatos
- **Primeros Auxilios** - Guía de emergencias felinas
- **Galería** - Colección de imágenes con filtros
- **Testimonios** - Slider con reseñas de usuarios
- **Contacto** - Formulario y datos de contacto

### Funcionalidades Interactivas
- Modo oscuro/claro con persistencia
- Cursor personalizado con estela de patitas
- Estadísticas animadas con contadores
- Acordeón expandible para información detallada
- Lightbox para ver imágenes en grande
- Slider automático de testimonios
- Filtros de galería por categoría
- Animaciones suaves al hacer scroll
- Diseño responsive para todos los dispositivos

## Cómo Usar

### Visualizar la página

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/gatocare.git
```

2. Abre `index.html` en tu navegador:
```bash
# En Windows
start index.html

# En Mac
open index.html

# En Linux
xdg-open index.html
```

3. ¡Disfruta de la página!

### Personalizar

#### Cambiar el tema de colores
Edita las variables CSS al inicio de `styles.css`:
```css
:root {
    --primary-color: #e67e22;    /* Color principal */
    --secondary-color: #d35400;  /* Color secundario */
    --accent-color: #f39c12;     /* Color de acento */
}
```

#### Actualizar datos de contacto
Edita la sección de contacto en `index.html`:
```html
<div class="contact-item">
    <i class="fas fa-phone"></i>
    <div>
        <h4>Teléfono</h4>
        <p>TU_NUMERO_AQUI</p>
    </div>
</div>
```

## Secciones Detalladas

### Calculadora de Edad
Convierte la edad de tu gato a años humanos usando la fórmula veterinaria actualizada:
- 1 año = 15 años humanos
- 2 años = 24 años humanos
- Cada año adicional = +4 años humanos

### Test de Conocimientos
Quiz interactivo con 6 preguntas sobre:
- Hábitos de sueño
- Sentidos felinos
- Nutrición
- Anatomía
- Comportamiento
- Salud

### Primeros Auxilios
Guía completa de emergencias:
- Envenenamiento
- Resucitación cardiopulmonar
- Fracturas y traumatismos
- Golpe de calor
- Problemas oculares
- Heridas y cortes

### Mitos vs Realidades
Desmiente creencias populares:
- Los gatos tienen nueve vidas
- Los gatos son antisociales
- La leche es buena para los gatos
- Los gatos siempre caen de pies
- Morder cables es por hambre
- Los gatos no necesitan vacunas
